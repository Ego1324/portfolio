Site vitrine CRM Web Laravel Sécurité

Infotools

Projet en deux missions : un site internet vitrine pour la présentation de l'entreprise, et une application CRM web réservée aux commerciaux avec authentification via l'Active Directory, gestion des rendez-vous et des clients.

🌐 Mission 3 – Site vitrine

  • Présentation de l'entreprise
  • Catalogue des produits phares (BDD)
  • Formulaire de demande de rendez-vous
  • Assignation RDV par le responsable
  • Validation côté serveur des formulaires
  • Journalisation des demandes (logs BDD)

💼 Mission 2 – CRM Web

  • Authentification via Active Directory (PHP)
  • Accès réservé aux commerciaux
  • Écran d'accueil avec rendez-vous à venir
  • Page produits et récapitulatif achats client
  • Tests unitaires (≥ 60 % couverture API)
  • Contrôle d'accès IDOR entre commerciaux

🛠️ Technologies utilisées

PHP Laravel MySQL HTML / CSS / JS Active Directory API REST Tests PHPUnit Blade (templates)

🔒 Exigences sécurité

  • Contrôle d'accès / IDOR (un commercial ne peut pas lire les données d'un autre)
  • Validation serveur — codes 422, rejet des injections / XSS
  • Limitation de débit (throttling) sur l'authentification
  • Journalisation des actions sensibles (logs d'audit BDD)
  • Au moins 3 tests automatisés Feature/HTTP (IDOR, validation, throttling)
