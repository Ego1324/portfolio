Infrastructure Réseau Linux Administration système

Héberge4

Plateforme d'hébergement web gratuit permettant à des particuliers de créer un compte et disposer automatiquement d'un site personnel. Le système gère de façon automatisée la création des comptes Linux, des entrées DNS, des configurations Apache et des bases de données.

⚙️ Fonctionnalités système

  • Inscription utilisateur avec alias unique (ex : alves71.hebergeX.lan)
  • Création automatique du compte Linux et du dossier public_html
  • Génération du fichier de config Apache par utilisateur
  • Ajout de l'entrée DNS dans Bind9 à l'inscription
  • Création de la base de données personnelle (MySQL)
  • Suppression complète du compte et de tous ses fichiers

🌐 Interface web PHP

  • Page d'inscription avec vérification d'alias
  • Authentification avec sessions PHP
  • Tableau de bord : espace utilisé, nb de pages
  • Modification des informations personnelles
  • Bouton de suppression de compte
  • Intégration GLPI pour tickets de support

🗄️ Services configurés

  • Serveur Web : Apache2 avec vhosts dynamiques
  • DNS : Bind9 avec zone hebergeX.lan
  • FTP : vsftpd pour upload des pages
  • BDD : MariaDB avec base par utilisateur
  • GLPI : ticketing et inventaire (FusionInventory)

💡 Compétences développées

  • Administration système Linux (Debian)
  • Automatisation par scripts Bash via PHP
  • Configuration réseau et DNS
  • Gestion de serveurs web multi-sites
  • Sécurité et gestion des droits (sudoers)

🛠️ Technologies utilisées

Linux Debian Apache2 PHP MySQL / MariaDB Bind9 (DNS) vsftpd (FTP) Bash GLPI
